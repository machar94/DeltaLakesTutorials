# pyspark Playground 

Playground for learning pyspark. 

### Prerequisites

* Poetry: 

    ```shell
    curl -sSL https://install.python-poetry.org | python3 -
    ```

* Java: 

    ```shell
    sudo apt install default-jre default-jdk
    ```

### Getting Started

1. Clone the repository

2. Download dependencies

    ```shell
    poetry init
    ```
